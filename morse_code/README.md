## Instructions

Write a ruby program which decodes a string of [International Morse
Code](http://en.wikipedia.org/wiki/International_Morse_Code).

## Sample Usage

### Input

```no-highlight
.-.. .- ..- -. -.-. .... / .- -.-. .- -.. . -- -.-- / .. ... / .-.. ---
-.-. .- - . -.. / .- - / ...-- ...-- / .... .- .-. .-. .. ... --- -. /
 .- ...- . .-.-.-
```

## Output

```no-highlight
"LAUNCH ACADEMY IS LOCATED AT 33 HARRISON AVE."
```
